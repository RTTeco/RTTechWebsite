Basic Instructions For Creating a Design for the Torchmate Plasma Metal Cutting System

Your design must be line copy. The software will not read or convert any shading or halftone textures.

Your design can be created in just about any art, or drawing program. Recommended programs are AutoCAD, PC Draft, and CorelDraw. Any design that has circles or arcs must be Converted to Curves.

Straight lines, squares, rectangles etc. can be converted as is.

The design, after it is created must be saved as a DXF file.

Lettering: When creating a design with lettering or text within CorelDraw, the text must be Converted to Curves From the Arrange Pull Down Menu. This will allow the software to transfer the meaning of those lines. The design, after it is created must be saved as a DXF file.

Basic Instructions For Using the Torchmate Plasma Metal Cutting System

Step 1:  Turn on computer, and Torchmate control box and power supply.

Step 2:  Open the Torchmate Application from the desktop by double clicking on it.

Step 3:  Choose “Yes Start Online” this connects the computer to the Torchmate controller. If you choose “No work offline” you can do your designing, but you will not be able to cut.

  Step 4: Choose “Agree” to the safety first written material. The main screen should appear.

  To import a document from Corel Draw, PCDraft, AutoCad, or Vector Cam, the document must be saved as a DXF file.

  Step 5:  Select Import DXF from the File pull down menu.

Find your file among the DXF files stored. My Documents, My Parts are where they should be stored. Double click on the file to select it, and then OK.

The Import setup menu will appear on your monitor.

Step 6:   Set proportional scale. This will enlarge or reduce your design.

1.0     Equals a 1” grid

2.0     Equals a 2” grid etc.

  Step 7:    Select Program Zero Location.

        XY of import will have the machine travel to the location of  your design.
         Lower left of tool path will move your design closer to the machines zero setting
         Choose Lower left of tool path, then OK.
            Your design will appear on the monitor.

Step 8:   Select Jog.

Select Speed. Choose Fast. This is for rapid movement only.

Move tool manually by mouse click and holding on the X and Y axis buttons on the monitor. Move the tool to the bottom left hand corner of the material you are going to cut.

Step 9: Select G Code. The status menu appears in red.

    Zero Machine Program by selecting “0”. Then OK.

    Zero Machine Relative by selecting “0”. Then OK.

    All coordinates should be zeroed. All coordinates must be zeroed.  

    Also it may be a good idea to Clear the torchmate under the Coordinates Menu.

Step 10: Set the height of the torch tip to about 1/8” or less above your material to be cut.

Note: If you would like to “air” cut you design. To make sure every thing is set continue with step 16, (Don’t turn on the plasma machine)

Step 11: Turn plasma cutting machine on. Set amperes to the highest setting.

Step 12: Make sure the material is grounded with the red ground clamp.

Step 13: Select speed to Slow, Medium, or Fast.

Set Feed Rate to the proper percentage. These two rating will depend upon the thickness of your material and the condition of the torch tip.

        Some common settings are as follows:

        1/8” > Slow > 50%

        3/8” > Slow > 12%

Step 14: Select Continuous (default)

Step 15: Select Start

By watching the monitor, the dark blue line is a feed (cut) and the dotted blue line represents a rapid movement (not cutting).

To do another cut of the same design, the machine needs to repositioned and be reset.
