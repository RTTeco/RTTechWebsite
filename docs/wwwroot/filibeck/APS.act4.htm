Activity 4 – Filters/Layers
(50 pts)  

  1. Use a picture of yourself, either an excising one, a scanned one or take a new digital one

    2.  Use the Lasso tool to cut yourself out and put on a new layer

    3.   Use Free Transform to resize yourself smaller or larger.

    4.   Apply a filter of your choice to the original layer.

    5.   Add a layer and color it.

    6.   Go to Save A Copy and make sure it is a JPEG file.  Insert into Publisher, resize if needed, type your name in a text box.

    7.   Print in color, and hand in.
