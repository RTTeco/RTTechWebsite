Putting a PowerPoint into FrontPage

     

  1. Open up Microsoft PowerPoint.  Find the presentation you want to use or create a new one.

 

  1. When you save the presentation go to File / Save as Web Page

 

  1. Check that the name is what you want or rename it.

 

  1. Save the presentation into your www folder.

 

  1. Open up Microsoft FrontPage. In the page you want the PowerPoint to play create a hyperlink to the PowerPoint file in your www folder.

 

  1. Preview in browser to make sure it works
