Minecraft EDU Infomation

Minecraft EDU is based on the Bedrock version of Minecraft. It has special features, but is limited in other ways. One large issue with Mincraft Education Edition is that it is currently impossible to create servers you can with both Java and Bedrock editions of Minecraft. To circumvent these problems, a hosted game can be created and left online at all times to act in a server fashion. I have done that for our district (really anyone with a K12 address and Minecraft Education Edition license can join). Two "servers" currently exist and are limited to 40 simultaneous players. Anarchy.RichardtonTaylorK12.tk is as the title states: an anarchy server where ther are no rules. Survival.RichardtonTaylorK12.tk is a survival world set on Easy. No griefing, stealing or PVP is allowed at any time.

    Anarchy

  • Set on Normal Difficulty
  • No Rules - PVP, Griefing, Stealing, Etc. is allowed. Pillage at your own risk!

    Survival

  • Set on Easy Difficulty
  • No PVP, Griefing, Stealing, Etc.

It should go without saying that all school rules in regards to conduct are to be followed per the student handbook. This also applies to the official school Discord server and Microsoft Teams group.
